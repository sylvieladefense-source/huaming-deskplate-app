# 华鸣智能桌牌 Web App

`web-app` 是用于 ESP8266 四色墨水屏桌牌的手机端网页 App。页面针对手机竖屏设计，可输入姓名、公司、职位，使用 HTML5 Canvas 实时预览固定尺寸 `240×416` 的黑、白、红、黄四色桌牌图像，并生成固件要求的 `24960 bytes` 二进制图像数据上传到设备。

## 文件说明

- `index.html`：手机端页面结构与表单。
- `style.css`：绿色科技感 UI 与竖屏适配样式。
- `app.js`：Canvas 实时绘制、四色量化、bin 打包和上传逻辑。
- `README.md`：使用说明。

## 使用方法

1. 将手机连接到 ESP8266 热点：`Huaming-Deskplate`。
2. 在浏览器打开本页面。
3. 输入姓名、公司、职位，预览会实时刷新。
4. 点击 **上传到桌牌**。
5. App 会通过 `POST` 上传到：`http://192.168.4.1/upload`。

上传使用 `multipart/form-data`，字段名固定为：`image`。

## Bin 格式

- 画布尺寸：`240×416`。
- 像素数量：`99840`。
- 色深：每像素 `2 bit`，每 4 个像素打包为 1 字节。
- 文件大小：`240 × 416 ÷ 4 = 24960 bytes`。
- 打包顺序：按行从左到右、从上到下，每字节高位到低位依次存放 4 个像素。
- 颜色编码：
  - `0`：白
  - `1`：黄
  - `2`：红
  - `3`：黑

## 与固件对应

当前固件的 `/upload` 接口会接收字段名为 `image` 的文件，并检查文件大小必须为 `24960 bytes`。上传成功后，ESP8266 会刷新墨水屏。
